# Template code capteur {sensor hat(B)}
Test programs for Sensor hat (B) (waveshare)
Exemple de code pour la lecture des capteurs disponible sur le sensor hat (B).  

## installation

Se placer dans le répertoire du projet.  
**Linux**  
```
python3 -m venv venv
source venv/bin/activate
venv/bin/pip install -r requirements.txt
```

**Widows**
```
python -m venv venv
venv\Scripts\activate
venv\Scripts\pip install -r requirements.txt
```

## Usage
Pour lancer le programme il suffit de l’exécuté :  
**Linux**
```
python3 main.py
```
**Windows**
```
python main.py
```

Pour stopper le programme, `CTL+C`
